E-commerce Admin Panel - Assets

This package contains:
- CSS for styling (`assets/css/style.css`)
- JavaScript for interactive features (`assets/js/script.js`)
- Placeholder folder for images (`assets/images/`)

To use:
1. Ensure your PHP files include:
   <link rel="stylesheet" href="assets/css/style.css">
   <script src="assets/js/script.js"></script>

2. Place your images inside `assets/images/`.

Enjoy your Admin Panel!
